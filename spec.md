# 研修課題５：出退勤打刻アプリ 仕様書

## 1. 開発目的
スマートフォンから直感的に操作できる出退勤管理システム。
LINE通知、スプレッドシート保存、PWA対応を組み合わせることで、実用性の高い業務ツールを目指す。

## 2. 実装機能
- **出勤打刻**: LINEへの【出勤】通知、スプレッドシートへの時刻保存。
- **退勤打刻**: 勤務時間の自動算出、LINEへの【退勤】通知、スプレッドシートの更新。
- **課題完了報告**: 管理者への最終報告通知、完了記録シートへの保存。
- **PWA**: ホーム画面への追加、ネイティブライクな動作。

## 3. 使用技術
- **フロント**: HTML5 / CSS3 (Vanilla) / JS (ES6+) / Lucide Icons
- **バックエンド**: Google Apps Script (Web App)
- **連携**: LINE Messaging API / Google Spreadsheet
- **ホスト**: GitHub Pages

## 4. UI設計方針
- **視認性**: 状況に応じたカラーリング（青＝出勤、赤＝退勤）。
- **プレミアム感**: ガラスモーフィズムとスムーズなアニメーション。
- **操作性**: スマホの親指で押しやすい「巨大ボタン（タップエリア拡張）」。

## 5. 課題設定・検証
- **LINEトークン**: YOZ7Uf... (設定済み)
- **グループID**: C5a5b3... (設定済み)
- **スプレッドシート構成**: 3シート（研修生マスタ、打刻記録、課題完了記録）
